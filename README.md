## Projects

### Featured: Cloud-Native Customer Segmentation
**[ML Pipeline Migration to BigQuery](https://github.com/CaballeroRAR/ds_projects/tree/main/1_ml-pipeline-migration-bigquery)**
Production-scalable migration of customer segmentation workflows to **Google Cloud BigQuery** using a **Medallion Architecture**.
- **Impact:** Detected significant segment drift between 2009 and 2011; achieved stable clustering separation verified via PCA.
- **Stack:** GCP (BigQuery ML), SQL ETL, Python (Loguru), Dimensionality Reduction (PCA).

### Machine Learning & Optimization
**[Gold Recovery Prediction](https://github.com/CaballeroRAR/ds_projects/tree/main/4_gold-recovery-industrial-optimization)**
Optimizing flotation efficiency in mining operations through regression modeling.
- **Results:** Achieved lowest sMAPE using ensemble methods; identified key reagent drivers.

**[Employee Salary Analysis – EDA & Clustering](https://github.com/CaballeroRAR/ds_projects/tree/main/3_employee_salary_eda_clustering)**
Workforce segmentation and compensation analysis using unsupervised learning.
- **Insights:** Identified 5 distinct employee clusters for targeted HR benchmarking.

### Analytics & Forecasting
**[Taxi Demand Forecasting](https://github.com/CaballeroRAR/ds_projects/tree/main/5_time-series-taxi-demand-forecasting)**
Predicting taxi order volume at airports using temporal decomposition and ML.
- **Outcome:** Captured complex daily/weekly seasonality with Gradient Boosting models.

**[A/B Test - E-commerce Website Optimization](https://github.com/CaballeroRAR/ds_projects/tree/main/2_ab-test-ecommerce-optimization)**
Statistical analysis of user conversion rates to drive design decisions.
- **Action:** Z-tests and P-value calculation to validate design hypothesis.
